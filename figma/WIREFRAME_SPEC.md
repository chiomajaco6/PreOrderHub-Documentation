# Figma Wireframe Specification for PreOrderHub

Create the following Figma pages:

1. Cover Page
2. Customer Storefront
3. Customer Checkout
4. Customer Dashboard
5. Admin Dashboard
6. Campaign Management
7. Payment Management
8. Mobile Screens

## Frame 1: Landing Page

Size: Desktop 1440 x 1024 and Mobile 390 x 844

Sections:

- Header: Logo, Search bar, Cart icon, Login button
- Hero banner: Campaign title, countdown timer, Shop Now button
- Categories: Shoes, Bags, Kitchen, Kids, Home, Fashion
- Product cards: Image, name, price, installment badge
- Preorder rules summary
- Footer

## Frame 2: Product Detail Page

Components:

- Product image gallery
- Product name
- Price without shipping fee
- Shipping fee notice: "Shipping fee will be communicated later"
- Size selector
- Colour selector
- Quantity selector
- Add to Cart button
- Installment Available badge

## Frame 3: Checkout Page

Components:

- Order summary
- Product total
- Minimum required deposit: 70%
- Amount customer wants to pay now
- Error state if amount is below 70%
- Preorder rules checkbox
- Place Order button

## Frame 4: Customer Dashboard

Cards:

- Total orders
- Total paid
- Outstanding balance
- Shipping fee balance
- Payment progress bar
- Shipment timeline
- Download receipt button

## Frame 5: Admin Dashboard

Cards:

- Active campaign
- Days remaining
- Total orders
- Total revenue
- Outstanding balances
- Customers owing shipping fee
- Recent payments
- Best-selling products

## Frame 6: Campaign Management

Fields:

- Campaign name
- Start date
- End date
- Campaign banner
- Status: upcoming, active, closed, completed
- Countdown preview
- Products uploaded
- Orders received

## Frame 7: Shipping Fee Management

Fields:

- Select campaign
- Select order or apply to all orders
- Enter shipping fee
- Update balances
- Notify customers

## Frame 8: Product Sourcing Request

Fields:

- Upload product image
- Product description
- Quantity
- Size
- Colour
- Budget
- Customer note
- Submit request

## Suggested Style

- Primary colour: Royal blue #2563EB
- Accent colour: Orange #F97316
- Background: #F8FAFC
- Success: #22C55E
- Warning: #F59E0B
- Error: #EF4444

## Mobile Bottom Navigation

Home | Search | Cart | Orders | Profile
